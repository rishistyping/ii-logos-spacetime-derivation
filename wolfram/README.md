# Wolfram Surface

This directory contains the public-facing Wolfram companion plan for *A Brief Derivation of Spacetime*.

Canonical surfaces:

- Builder source: `build_spacetime_notebook.wl`
- Frozen notebook contract: `notebook_plan.md`
- Generated notebook target: `notebooks/spacetime_explainer.nb`
- Future preview assets: `assets/`

## Purpose

The Wolfram workbook is a computational companion, not proof authority. It should make the paper's matrix spine easy to inspect:

- the six-step derivation funnel
- the exact `A(λ)` time-evolution matrix
- trace, determinant, characteristic polynomial, and square
- branchwise phase portraits for `λ > 0`, `λ = 0`, and `λ < 0`
- the boundary between computed panels, Lean targets, imported theorem interfaces, and interpretation

## Build

```bash
wolframscript -file wolfram/build_spacetime_notebook.wl
```

The v0 builder is a placeholder contract. The reproducible matrix artifacts are generated by the SymPy scripts under `sympy/`.
